Build and import dynamic lib file
----------------------------------------------

1.) Get official wolfSSL sourcecode from: https://github.com/wolfSSL/wolfssl
2.) Add "__declspec(dllexport)" infront of return type of function to export (ssl.h)
3.) General properties of "wolfssl" project -> Configuration Type = Dynamic Library (.dll)
4.) Build "wolfssl"
5.) Open Visual Studio prompt and navigate to built wolfssl.dll
6.) dumpbin /exports wolfssl.dll > wolfssl.def
7.) Change text in wolf.def file to:
	EXPORTS
	func_name1...
	func_name2...
	func_name3...
8.) lib /def:wolfssl.def /out:wolfssl.lib /machine:x86
9.) Copy wolfssl.dll to client.exe locations
10.) Include .lib into Visual studio (Project properties -> Linker -> Input -> Additional Dependencies
